CRUD Overgrowth

 Welcome to Crud Overgrowth!

Crud Overgrowth is a full-stack, stateful web app that draws functionality and design inspiration from stackoverflow.com. This application is supported by a React/Redux frontend and a Flask Python backend, which uses SQLAlchemy to interact with a PostgreSql database. The current build of Crud Overgrowth offers two key features of interaction: questions and answers. Users are able to sign up, log in, and ask programming-related questions, as well as answer the questions of other users, all with rich text formatting.

Check out the Crud Overgrowth repo!

Scuttlr

 A sample image post for Scuttlr

Scuttlr is fullstack, stateful web-app that draws functionality and style inspiration from tumblr.com, developed and deployed in one week in collaboration with a small team. This application is supported by a React/Redux frontend and a Flask Python backend, which uses SQLAlchemy to interact with a PostgreSql database. The current build of Scuttlr offers four key features of interaction: posts, comments, likes, and follows. Users are able to switch between the all-posts feed and a personal feed. A user's personal feed is influenced by the users that they follow. When logged in, users are able to like and make comments on any post.

Check out the Scuttlr repo!

NateBnB

 The NateBnB landing page

NateBnB is my first full-stack solo project, which draws design and functionality from AirBnB. This application is supported with a React/Redux frontend and an Express.js backend, which uses Sequelize to interact with a PostgresSql database. NateBnB allows users to create and post listings and reviews of other users' listings, all while providing a smooth and intuitive user experience.

Check out the NateBnB repo!
